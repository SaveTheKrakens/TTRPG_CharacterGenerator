# TTRPG Character Generator

Welcome to the TTRPG Character Generator!  
This project is designed to help users quickly create and manage characters for tabletop role-playing games (TTRPGs).

## Features

- Random character generation (WIP)
- Customizable attributes and stats (WIP, currently implementing sql in forms)
- Support for multiple TTRPG systems (WIP, currently just Dnd 5e)
- Save, load, and export character sheets (WIP, waiting for customizing completion)

## Getting Started

### Prerequisites

- The current project is not build ready, download the project .zip file and open the solution in Visual Studio 2022. You may need to download the SQLite extensions in order for the database to work.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/SaveTheKrakens/TTRPG_CharacterGenerator.git
   ```
2. Install dependencies:
   SQLite and SQL Server Compact Toolbox in Visual Studio > Extensions > Manage Extensions

### Usage

- Run the character generator from the main branch.
